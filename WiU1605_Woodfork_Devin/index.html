Project Name: WorldSurf

Project Description

This project will include a series of wireframes that will test and display my ability to create user friendly designs for the web. I will display my skills obtained by properly using: Visibility, Consistency, Constraints, Feedback, Errors, Affordances and much more. I will upload my wireframes to NavFlow to create usability tests which will determine how user friendly my design is. I will post the results and links to this landing page and upload it to the gh-pages branch of my GitHub account.This landing page will be styled using CSS and my choice of image. I will test the HTML and CSS to ensure it is valid.
